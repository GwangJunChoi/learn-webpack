# webpack dev Middleware 실습
>##### package.js 생성
```
npm init -y
```
>##### webpack & express & webpack-dev-middleware 설치
```
npm install webpack --save
npm install express webpack-dev-middleware --save-dev
```
>##### ejs 템플릿 엔진 설치
```
npm install ejs --save
```
>##### 서버 실행
```
node server.js
```
